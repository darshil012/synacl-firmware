# Synacl Firmware

Open-source firmware for connecting ESP32 and compatible microcontrollers to the [Synacl](https://synacl.com) IoT platform.

## Coming Soon

We're cleaning things up before the public release. Star or watch this repo to be notified when it drops.

What to expect:

- **MQTT + HTTP + MODBUS** — native connectivity to the Synacl platform out of the box
- **Wi-Fi provisioning** — BLE-assisted setup, no hardcoded credentials
- **OTA updates** — pull firmware updates from the platform
- **ESP32 / ESP32-S3** — primary targets, with a clean HAL for other platforms
